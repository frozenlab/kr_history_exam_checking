# kr_history_exam_checking


한국사능력검정시험 채점기 파일
 회차 : 11회 ~ 78회
 선택 가능한 시험 난이도 : 고급, 중급, 초급, 심화, 기본
 고급, 중급, 초급은 47회 개편 이전 난이도
 심화와 기본은 47회부터 적용된 현재 실시중인 난이도

엑셀 파일 설명
 정답 시트 - 각 시험의 문제의 정답들
 배점 시트 - 각 시험의 문제의 배점들
 시험정보 시트 - 시험에 대한 정보들
 시험일 - 시험이 시행된 날
    문항수 - 해당 시험의 문제 개수
    선택지수 - 해당 시험의 선택지 개수(5지선다형, 4지선다형)
    응시자수 - 해당 시험을 치룬 사람의 수
    합격자수 - 해당 시험에 합격한 사람의 수
    합격률 - 합격자 수 / 응시자 수
  등급통계 시트 - 48회 부터 제공된 각 등급별 합격자수
    급수 - 해당 시험으로 될 수 있는 급수들 (심화 : 1,2,3급, 기본 : 4,5,6급)
    합격자수 - 해당 급수에 합격한 사람의 수
